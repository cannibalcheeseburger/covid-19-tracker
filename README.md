 ## Installation

 - [Python 3.7.6](https://www.python.org/downloads/)

 - Build from source
 
```
git clone https://github.com/cannibalcheeseburger/covid-19-tracker.git
cd covid-19-tracker
```

 - Requirements
```
python -m pip install -r requirements.txt
```

 - Running program
 ```
python app.py
 ```
